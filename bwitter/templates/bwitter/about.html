About Bwitter

An open-source CRUD Application developed by Jermaine Mercado.

This web application was created during the midst of the COVID-19 pandemic to develop a knowledge of Python and the Django web-framework. Front-end styling via Bootstrap CSS. Things learned during the development of the project are the use of class-based views, database connections, SQLite models, URL routing, CSRF protection, SMTP server set-up, user authentication, and Django's built in administrator functionalities. For more information, visit the open-source git repository containing the project's source.
